動詞變化完全攻略

日語動詞變化是最重要也最令人頭痛的文法基礎。本頁涵蓋三類動詞的所有變化形、て形音變規則、陷阱動詞列表，以及互動練習工具。掌握這些，N2 文法就有了穩固的基礎。

1 動詞分類（三大類）

II 一段動詞（Group 2 / ichidan / 上一段・下一段）
辨別規則：字尾一定是「る」，且「る」前面的假名在「い段」或「え段」（i/e 母音行）。

變化方式最簡單：去掉「る」，直接加上各種語尾。

動詞去るます形ない形
食べる 食べ食べます食べない
見る 見見ます見ない
起きる 起き起きます起きない
教える 教え教えます教えない
I 五段動詞（Group 1 / godan）
辨別規則：字尾不是「る」的動詞，或者字尾是「る」但前面不在い段/え段。也包含所有陷阱動詞。

五段動詞的語幹最後一個假名會在「あいうえお」五段之間變化（所以叫五段）。

五段子音行對應：
く行：か き く け こ ｜ ぐ行：が ぎ ぐ げ ご
す行：さ し す せ そ ｜ つ行：た ち つ て と
ぬ行：な に ぬ ね の ｜ ぶ行：ば び ぶ べ ぼ
む行：ま み む め も ｜ る行：ら り る れ ろ
う行：（わ） い う え お
動詞行ます形 (い段)ない形 (あ段)
書く か行書きます書かない
飲む ま行飲みます飲まない
話す さ行話します話さない
買う わ行買います買わない
待つ た行待ちます待たない
死ぬ な行死にます死なない
遊ぶ ば行遊びます遊ばない
走る ら行走ります走らない
泳ぐ が行泳ぎます泳がない
III 不規則動詞（Group 3 / irregular）
日語只有兩個不規則動詞：する（做）和来る（くる）（來）。但「する」可以和名詞結合（勉強する、運動する），所以實際上數量很多。
動詞ます形ない形て形た形
する しますしないしてした
来る 来ます（きます）来ない（こない）来て（きて）来た（きた）
注意「来る」的讀音變化！
辞書形：くる ／ ます形：きます ／ ない形：こない ／ て形：きて
母音在 く・き・こ 之間切換，這就是不規則的地方。

3 て形音變規則（最重要！）

て形是日語最常用的變化形之一，用於連接句子、請求、進行式等。五段動詞的て形有音便（おんびん）規則，必須牢記。

核 五段動詞 て形 五大音變規則

1. イ音便：く → いて

書く → 書いて、聞く → 聞いて、歩く → 歩いて

例外！ 行く → 行って（不是行いて！這是唯一的例外）

2. イ音便（濁音）：ぐ → いで

泳ぐ → 泳いで、脱ぐ → 脱いで、急ぐ → 急いで

3. す → して

話す → 話して、出す → 出して、消す → 消して

4. 撥音便：む・ぶ・ぬ → んで

飲む → 飲んで、遊ぶ → 遊んで、死ぬ → 死んで

5. 促音便：う・つ・る → って

買う → 買って、待つ → 待って、走る → 走って

一段動詞和不規則動詞不需要音變：
一段：去る＋て → 食べて、見て、起きて
不規則：する→して、来る→きて
記 記憶口訣
「一口氣記住て形！」

く → いて ぐ → いで ← 清音/濁音是一對
す → して ← す就是し（同行）
む・ぶ・ぬ → んで ← 嘴巴閉起來的音（m/b/n）變「ん」
う・つ・る → って ← 其餘全部變「っ」促音

口訣：「く・ぐ 一對，す 獨行，嘴閉（む・ぶ・ぬ）撥音，剩下促音」

日語歌訣（配「きらきら星」旋律）：
いて、いで、して、んで、って〜
く、ぐ、す、む・ぶ・ぬ、う・つ・る〜
た形和て形規則完全一樣！只需把「て→た」「で→だ」：
書いて → 書いた、泳いで → 泳いだ、飲んで → 飲んだ、買って → 買った
練 て形練習例句
用法例句中文

進行式今、本を読んでいます。現在正在看書。
連接朝ごはんを食べて、学校に行きます。吃完早餐去學校。
許可ここで写真を撮ってもいいですか。可以在這裡拍照嗎？

5 自動詞・他動詞對照表

日語中很多動詞成對出現：自動詞（不及物）描述事物自然發生，他動詞（及物）描述人為動作。

記憶技巧：很多時候，含「〜ある」的是自動詞，含「〜える」的是他動詞。例：集まる（自）/ 集める（他）。但也有例外，需要逐對記憶。
